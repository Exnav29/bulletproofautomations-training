This Is the Starting Point, Not the Finish Line

Completing the beginner cohort does not make someone a professional automation builder.

It gives the foundation needed to continue into intermediate, advanced, and use-case cohorts. The full pathway is what prepares someone to be considered for more serious work.

What You Will Learn

What n8n Is

Where it fits, when to use it, and how to think about workflows.

Local Setup

Local n8n setup with Docker for continued practice.

Workflow Basics

Triggers, nodes, connections, and executions.

Working With Data

Move fields between nodes and understand execution output.

Troubleshooting

Read errors, rerun steps, and fix beginner mistakes.

Safe Access

Credentials and safe tool access basics.

Useful Workflows

Build simple automations that do practical work.

Export and Document

Export workflow JSON and explain what was built.

8-Week Beginner Structure

The exact outline may be adjusted as cohort needs become clearer.

Week 1

Orientation and First Workflow

Local setup expectations, what n8n is, and the first working automation.

Week 2

Triggers and Actions

Workflow structure, node connections, and basic execution flow.

Week 3

Data Between Nodes

Reading outputs, mapping fields, and keeping data understandable.

Week 4

Conditions and Logic

Branching, filters, and beginner workflow decisions.

Week 5

HTTP and APIs

HTTP requests, APIs, and external services at a beginner level.

Week 6

Credentials and Safety

Credentials, security basics, and safe workflow habits.

Week 7

Testing and Documentation

Troubleshooting, testing, and documenting workflows clearly.

Week 8

Final Beginner Project

Portfolio packaging, workflow export, and next-step review.

Prerequisites Before the First Saturday Class

Be Ready Before Saturday

  • Participants must have access to n8n before the first Saturday class.
  • Saturday class will not be used for Docker installation, n8n installation, or individual computer setup.
  • Participants who are not ready can still watch, but may not be able to follow along hands-on.
Option A: Recommended

Local n8n with Docker

  • Install Docker Desktop
  • Run n8n locally
  • Best for continued practice
  • No dependency on paying for cloud after a trial
Option B: Fallback

n8n Cloud Trial

  • Useful if local setup fails
  • Trial is temporary
  • May require payment after the trial period
  • Not the preferred long-term option for free practice

Thursday Setup Pre-Session

  • 30-minute optional but strongly recommended pre-session
  • Designed to help people understand local setup
  • Recorded and posted to YouTube
  • If someone cannot attend Thursday, they should watch the recording before Saturday
  • Individual troubleshooting may be limited
  • The purpose is preparation, not a full support clinic

Assignments and Portfolio Work

  • Every Saturday class will have an assignment
  • Participants should save exported workflow JSON files when requested
  • Participants may be asked to submit screenshots or short explanations
  • The goal is to build proof of work, not just attend calls

Certificate of Completion Requirements

  • Attend at least 80% of sessions
  • Complete all required assignments
  • Submit required evidence of work when requested
  • Follow class expectations

This is a Certificate of Completion, not an official n8n credential.

Class Expectations

  • Show up on time
  • Come prepared
  • Complete prerequisites before class
  • Do the assignments
  • Ask questions respectfully
  • Do not expect Saturday class to become setup troubleshooting
  • Treat this like skill development, not entertainment
